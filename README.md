
# 🏆 FIFA World Cup 2026 - API REST (Partie 2 : Sécurisation)

Cette API REST professionnelle, développée avec **Node.js, Express et Sequelize (PostgreSQL)**, permet de gérer les arbitres, les matchs et leurs affectations pour la Coupe du Monde 2026. La **Partie 2** ajoute une couche de sécurité complète avec authentification **JWT (JSON Web Tokens)** et contrôle d'accès basé sur les rôles **(RBAC)**.

---

## 🚀 Configuration et Installation

### 1. Installer les dépendances

```bash
npm install
```
